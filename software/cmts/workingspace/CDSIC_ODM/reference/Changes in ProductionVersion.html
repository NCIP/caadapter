Changes in Production Version:

  • ValueListDef and ValueListRef are not included.
  • FormalExpression added under RangeCheck
  • Support for URL data types.
  • Eliminated support for mixing ItemData and ItemData[TYPE] elements in a single ItemGroupData element.
  • Description added to Protocol element.
  • AnnotationGroup, AuditRecordGroup, SignatureGroup element names changed to Annotations, AuditRecords, Signatures respectively.
