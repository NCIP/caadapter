caAdapter Translational Data Mapping Service (TDMS)
Version 4.0 TDMS 1.0

caAdapter is an open source tool set that provides model mapping services in support of caCORE components 
and facilitates data mapping and transformation among different kinds of data sources.

This release only contains the TDMS module. The TDMS provides mapping capability from a CSV metadata to 
an object model or a data model. To use other caAdapter capabilities, please use caAdapter tools suite or 
other individual services of caAdapter.
